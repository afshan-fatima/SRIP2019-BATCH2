Projects

  • STUDENT REPORT CARD
    • This project reduces the manual work of keeping the students record in a convenient way and provides an easy way to display the student’s personal details, attendance, marks etc.
  • SMARTPHONE CONTROLLED RC CAR
    • In this project robot car is controlled using an Android smartphone with ArduinoRC application installed in it. This car moves in the direction according to the command given from smartphone with the help of a Wi-Fi module Node MCU.It can be also used in home automation system.
  • SPORTS CLUB
    • In this project a website is created for our college where the student achievements and other records are displayed including coach and faculty profiles. It also displays the gallery of the events conducted which reduces the manual work of keeping records and provides an efficient way to access it.
